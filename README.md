# Projet Compost IA

Ce projet, nommé *Compost IA, est réalisé dans le cadre d'un cursus à l'*[INSA Rouen Normandie](https://www.insa-rouen.fr/)*.

L'objectif est de développer un modèle de Deep Learning basé sur *YOLO (You Only Look Once)* pour un système de station de tri de déchets. Le modèle est entraîné pour classifier automatiquement les déchets en quatre catégories afin d'isoler ce qui est compostable :

* *Compostable* (nom de code : Compsot)
* *Non Compostable* (nom de code : NonCompsot)
* *Dangereux* (nom de code : Dgrx)
* *Mécaniquement Risqué* (nom de code : Mrisq)
